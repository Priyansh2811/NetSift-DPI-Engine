# 🧠 NetSift DPI Engine

A High-Performance, Multi-Threaded Deep Packet Inspection (DPI) & Network Analytics Dashboard built with C++17, Python Flask, and Tailwind CSS. NetSift processes raw binary PCAP logs, executes parallel parsing, decodes Layer 7 TLS SNI signatures, and renders real-time bandwidth telemetry.

## ✨ Features

- **Deep Packet Inspection (L7 Extraction):** Decodes explicit domain mappings directly from raw cryptographic TLS Handshake bytes.
- **Dynamic Bandwidth Timeline Spikes:** Tracks and visualizes real-time packet flow velocity over sequential 10-second axis parameters.
- **Fault-Tolerant Simulation Framework:** Automatically injects high-fidelity matrix telemetry on zero-signature samples to prevent interface crashes.
- **Real-Time Client Searching:** Instant 0ms client-side fuzzy searching, table sorting, and dropdown criteria protocol filtering.
- **Corporate Report Exporter:** Direct translation of JavaScript runtime state memory into downloadable Microsoft Excel-ready CSV sheets.

## 📸 Sections Included

- **Metrics Analytics Counters:** Dynamic widgets tracking Total Volume Bytes, Packet Counts, and TCP/UDP distribution rules.
- **Traffic Bandwidth Over Time:** A fluid neon line chart mapping continuous packet delivery velocity.
- **Protocol Distribution Matrix:** Interactive doughnut charts tracking custom application signatures.
- **L7 Extracted SNI Table:** Responsive data grid with integrated search inputs and quick protocol filters.
